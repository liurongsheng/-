# 颜色选择

### 四色选择
```html
<style>
.colors {
  color：#eab50b;
  color：#d01717;
  color：#9dd61d;
  color：#338dc5;
}
</style>
```

### 按钮置灰颜色
```html
<style>
.form-data .lang-btn.off{
  color: #a0a0a0;
  background-color: #e5e5e5;
}
</style>
```


## 常用颜色
```html
<style>
.colors {
  color: #42a5f5; // 蓝色
}
</style>
```
